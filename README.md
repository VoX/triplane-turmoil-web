# Triplane Turmoil — Web Remake

Browser-playable remake of [Triplane Turmoil](https://en.wikipedia.org/wiki/Triplane_Turmoil)'s multiplayer dogfight mode.

Reference implementation: [sergiou87/triplane-turmoil](https://github.com/sergiou87/triplane-turmoil) (C++/SDL2, GPLv3).

## Status
M0 Bootstrap. See [PRODUCT_PLAN.md](./PRODUCT_PLAN.md).

## Local dev
```
npm install
npm run dev
```

Arrow keys: left/right turn, up/down throttle.
